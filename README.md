### Do you think you know your friends well? Try solving them in a puzzle! ###

#### The game has the following features: ####
- Users have the freedom to choose their own image for the puzzle.
- Users can have a mini display of original image to get a clue of how to solve the image.
- Users can choose difficulty by setting the number of rows and columns allowed for their puzzle.
#### Open maven project using your favorite IDE and run the GamePlay class to start playing. Enjoy! ####
